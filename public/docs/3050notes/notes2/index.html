Random process

sample space: sampling produces a sequence: time sqeuence.

fix the time point: t0: X(s1, t0), the function of the sample and time. That gives a random variable.

Then omit s1 here. X(t0).

Define:

  • Mean: E[X(t)] = ∫fX(t)(x)xdx

  • Auto-correlation: Rx(t1, t2) = E[X(t1)X(t2)] = ∫∫fX(t1)X(t2)(x1, x2)x1x2dx1dx2. conjugate of X2 when complex number involved.

  • wide sense stationary (WSS): μx = μ(t), Rx(t1, t2) = Rx(t2 − t1) (only the difference matters, shift!)

    In other words, the process can be viewed as a kind of time-invariance.

    For wide sense stationary which is a little weak than the strict stationary since it only requires the first moment (mean value) and autocovariance remain unchanged with respect to the time & second moment is finite. (TODO: an example of second moment?)

P.S. The autocovariance: which shows the combined variance of multiple variables: the changing tendencies are the same for the two variables: positive, if opposite: negative.

For stationary, the autocovariance is:

\begin{flalign} Cx(t1, t2)  = E[(X(t1) − μx)(X(t2) − μx)]      = E(X(t1)X(t2)) − μx(E(X(t1)) + E(X(t2))) + μ2x      = Rx(t1, t2) − μ2x\endflalign\endalign*

So the autocorelation function and mean value can determine the autocovariance for the stationary process.

PCM (Pulse Coded Modulation)

In PAM the g(t) takes continuous values on discrete time.

Introduce the Quantization to reduce the demodulation error, while brings the quantization error.

Idea: approximate infinite number of analog values to finite number of values.

Quantization

The quantization process is accomplished by applying an step function to the message.

step size:

Δ = (2max{\absm(t)})/(L)

where L is the number of levels. The error will be smaller than the half of the step size:

\abse(n) ≤ (Δ)/(2) = (max{\absm(t)})/(L)

Advantage: robust to noises

Encoding

Finite possibilities: encode using bits.

The number of bits: n = celling(log2(L)) .

Methods

  1. Unipolar NRZ (non-return-to-zero)

    range from 0 to 1, with no return gap between binary values.

    Drawback: (with bipolar NRZ): mean > 0, has nonzero DC term.

  2. Bipolar NRZ

    range from -1 to 1, -1 denotes 0, and no return to 0.

    Mean = 0

  3. Unipolar RZ

    Easier for synchronization than NRZ. Larger bandwidth is required.

Overall process

Sender

m(t) → LPF → Sampling → Quantization → Encoding → PCM waveform

Receiver

 → Decoder → Reconstruction LPF → m̂(t)

Bit interval : (Ts)/(R) = T, the bandwidth is approximately (1)/(T).

Data rate: (1)/(T) bits/sec.

bandwidth of m(t) is 3.1kHz.

sampling frequency fs = 8kHz, which gives Ts = 1 ⁄ 8k = 125μs.

data rate: rate = 1.544Mbps
